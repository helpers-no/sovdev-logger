# sovdev-logger

Multi-language structured logging with zero-effort observability.

Write one log entry and automatically get structured logs, metrics
dashboards, and distributed traces — correlated automatically, and
compatible with any OpenTelemetry backend (Azure Monitor, Grafana Cloud,
Datadog, New Relic, Honeycomb, or self-hosted infrastructure).

Currently available for TypeScript, with Go, Python, C#, Rust, and PHP
planned.

This docs site is newly set up — content is being migrated in from the
repo's [`specification/`](https://github.com/helpers-no/sovdev-logger/tree/main/specification)
and [`docs/`](https://github.com/helpers-no/sovdev-logger/tree/main/docs)
folders. In the meantime, see the
[project README](https://github.com/helpers-no/sovdev-logger#readme) for
the full picture.
